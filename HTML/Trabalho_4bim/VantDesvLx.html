Vantagens e Desvantagens de usar Linux






Vantagens





Pode ser executado com recursos mínimos do sistema



Quando os usuários pensam em tentar outro sistema operacional, é porque estão frustrados com o desempenho de seu sistema, ou para reviver um antigo laptop ou um sistema que está defasado.

As distribuições Linux rodam em hardware decente. É desnecessário ter o mais recente e melhor. Além disso, existem distribuições Linux leves para rodar em hardware mais antigo, sem problemas.

Assim, há maiores chances de reviver um sistema antigo ou obter um computador de desempenho rápido em pouco tempo, com o Linux.






Estabilidade



Muitos servidores Linux passam anos ou toda a sua vida útil sem a necessidade de reiniciar, ou seja 100% do tempo em funcionamento, ou uptime.






Documentação



A documentação no Linux é abrangente e aborda uma ampla variedade de tópicos relacionados ao sistema operacional, software, programação, administração de sistemas e muito mais. Tudo no Linux é documentado e pode ser alterado ou modificado para ajudar a cumprir uma tarefa de modo mais eficiente possível.






Custo



O Linux é um sistema operacional de código aberto, o que significa que ele é gratuito de se baixar e usar. Isso permite que as empresas economizem dinheiro em licenças de software e manutenção.






Menos exposto a malware



Nenhum sistema operacional está a salvo de arquivos ou scripts maliciosos. Se você baixar e executar algo de uma fonte desconhecida, não poderá garantir sua segurança. No entanto, as coisas são melhores para o Linux. No desktop ainda não é algo para se preocupar.

Atores maliciosos visam plataformas que são mais populares, e o Linux não tem uma grande participação no espaço de desktop, para atrair esse tipo de atenção. Tudo o que você precisa fazer é seguir os pacotes de software oficiais e ler as instruções antes de fazer qualquer coisa.

Como uma vantagem extra, você não precisa necessariamente de um programa antivírus para obter proteção contra malware.






Costumização



Com um código-fonte aberto, você tem a liberdade de personalizar sua experiência Linux o quanto quiser.

Claro, necessita-se de um pouco de conhecimento técnico para usá-lo melhor. Mesmo sem nenhuma experiência, obtém-se mais recursos de personalização em seu sistema operacional, quando comparado ao macOS e ao Windows.

Quem gosta de personalizar sua experiência e está disposto a fazer um esforço extra, consulte o guia de personalização do KDE e as opções de encaixe para obter ideias básicas.






Algo para todos



Com o macOS ou Windows, você fica limitado às opções de design/preferência finalizadas pela Microsoft ou pela Apple. No Linux, há várias distribuições Linux com focos distintos.

Você pode obter algo que parece bonito fora da caixa ou algo que oferece opções de personalização malucas. Você não ficará sem opções com o Linux. Pode-se começar com as opções que oferecem a melhor experiência do usuário.






Ambiente de Desenvolvimento Completo



Para um desenvolvedor de software ou estudante aprendendo a codificar, o Linux definitivamente tem uma vantagem: muitas das ferramentas de compilação estão disponíveis e integradas ao Linux. Com o Docker, você pode criar facilmente um ambiente de teste especializado.

A Microsoft sabe sobre essa parte e é por isso que criou o WSL, dando aos desenvolvedores acesso a ambientes Linux dentro do Windows. Ainda assim, o WSL não chega nem perto da experiência real do Linux. O mesmo vale para usar o Docker no Windows.

O mesmo não pode ser dito sobre web design, porque as cobiçadas ferramentas da Adobe ainda não estão disponíveis no Linux. Mas se você não precisa da Adobe para o seu trabalho, o Linux é uma boa escolha.






Aprender Linux é uma habilidade que você deve ter



Há uma curva de aprendizado para usar o Linux, mas ele fornece informações sobre várias coisas. Você aprende como as coisas funcionam, em um sistema operacional, explorando-o e personalizando-o, ou apenas usando-o.

Nem todo mundo sabe como usar o Linux. Portanto, pode ser uma grande habilidade ganhar e expandir seu conhecimento de software e computadores.






Linux é uma habilidade de trabalho em demanda



Você pode trabalhar para se tornar um administrador de sistema Linux ou um especialista em segurança e preencher várias outras funções de trabalho, aprendendo os fundamentos do Linux. Então, aprender Linux abre uma série de oportunidades.






Amigável à privacidade



Atualmente, não se pode usar o Windows sem uma conta da Microsoft. E, ao configurar o Windows, você descobrirá que ele tenta rastrear os dados de vários serviços e aplicativos, desrespeitando sua privacidade por padrão.

Esse não é o caso do Linux. Embora alguns aplicativos/distribuições possam ter um recurso opcional para compartilhar informações úteis com eles, isso nunca foi um grande problema. A maioria das coisas oferecem privacidade máxima por padrão, sem a necessidade de configurar nada.

A Apple e a Microsoft, por outro lado, têm táticas inteligentes para coletar dados do seu computador. Ocasionalmente, eles registram sua atividade na loja de aplicativos e, enquanto você está conectado, por meio de sua conta.






Desvantagens





Curva de aprendizado



De vez em quando, não se trata apenas de aprender uma nova habilidade, mas de sentir-se confortável o mais rápido possível. Se um usuário não conseguir realizar a tarefa que pretende fazer, não é para ele. Isso vale para todos os sistemas operacionais.

Alguns usuários entendem mais rápido do que outros. Mas, em geral, quando você entra no mundo Linux, você precisa estar disposto a se esforçar um pouco, para aprender as coisas que não são óbvias.






Variedade



Escolher uma das melhores distribuições Linux adaptadas para iniciantes pode ser difícil no início.






Participação de mercado no espaço de desktop



Linux não é um sistema operacional de desktop popular. Isso não é motivo de preocupação. No entanto, sem ter uma presença significativa no mercado, você não pode esperar que os desenvolvedores de aplicativos criem/mantenham ferramentas para Linux.

Claro, existem muitas ferramentas essenciais e populares disponíveis. Mas continua sendo um fator que pode significar que nem todas as boas ferramentas/serviços funcionam no pinguim.






Falta de software proprietário



Nem todos estão interessados em trazer suas ferramentas/aplicativos para o Linux. Você pode não encontrar todas as boas ofertas proprietárias para Windows/macOS. É possível usar uma camada de compatibilidade para executar programas destes SO’s. Mas isso não funciona o tempo todo. Por exemplo, não há suporte oficial do Microsoft 365 para Linux e ferramentas como o Wallpaper Engine.






Não é um sistema operacional voltado para jogos



Se você quiser jogar em seu computador, o Windows continua sendo a melhor opção para oferecer suporte aos mais novos hardwares e tecnologias.

Quando se trata de Linux, existem muitos “se e mas” para uma resposta clara. Você pode consultar o guia de jogos para Linux para explorar mais, se estiver interessado.






Falta de suporte técnico profissional



Existem opções de suporte técnico que podem orientar os usuários/corrigir problemas remotamente em seu laptop ou computador. Pode-se buscar ajuda da comunidade, mas nem sempre é simples, quanto alguns serviços profissionais de suporte técnico.
